# Task 3 - Sales Data Analysis Using SQL

## Objective

Analyze sales data using SQL queries to identify sales trends, customer behavior, payment preferences, and product performance.

## Tools Used

* MySQL Workbench
* SQL

## Dataset

Sales dataset containing 1200 records with customer, product, payment, and order information.

## Analysis Performed

* Product-wise Total Sales
* Order Status Analysis
* Payment Method Analysis
* Top 5 Customers by Spending
* Dataset Preview
* First 10 Records
* Total Records Count
* Referral Source Analysis
* Average Product Price Analysis
* Product-wise Quantity Sold Analysis

## Key Findings

* Chair generated the highest total sales.
* Credit Card was the highest revenue-generating payment method.
* Customer C38840 spent the most.
* Instagram generated the highest referral sales.
* The dataset contains 1200 records.

## Outcome

Used SQL aggregation, grouping, sorting, and filtering techniques to extract meaningful business insights from sales data.
